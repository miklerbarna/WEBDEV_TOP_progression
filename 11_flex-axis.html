Flexboxes can work vertically or horizontally. Flexboxes always have two axes: The main axis and the cross axis. flex-direction: row; (the default) puts the main axis horizontal and the cross axis vertical

In the first example below, we put display: flex (which is the same as flex-direction: row;) on a div and it arranged its children horizontally. Try changing it to flex-direction: column; it will arrange the children vertically.

Here we can try out Alignments On the parent div, try justify-content: center; justify-content: start; (the default) justify-content: end; justify-content: space-evenly; This tells the container, how its children should be placed on the main axis if there is available space (Hence the flex-grow: 0; property)

To manipulate flex items on the cross axis, we use the align-items property in the flex container Try align-items: stretch;(default), align-items: flex-start; align-items: center; align-items: flex-end; align-items: baseline;(The first line of text for all items will align on the same baseline.)
